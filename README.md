## Getting Started

### Prerequisites

- Node.js >= 18.0.0
- npm or yarn
- MetaMask wallet

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Deployment
